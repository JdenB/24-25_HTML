Deku/Green Hero

Green Hero is the first character to be ever developed in Heroes Battlegrounds and the easiest to use in general. Green Hero is a free in-game character and the default when playing the game for the first time. Green Hero has decent combo potential and a deadly ultimate when used correctly.
